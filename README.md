Install dependecies Client

```javascript
// Install adonis cli
npm i -g @adonisjs/cli

// Enter in folder
cd server/

// Configure .env in project

// Migrations for BD
adonis migration:run

// Run Project
adonis serve

// Run project in developer mode
adonis serve --dev

// PORT 3333
```

Install dependecies Client

```javascript
// Enter in folder
cd client/

// Install dependecies
yarn install

// Run Project
yarn start

// PORT 3000
```
